# 开头（蛇棒）

程序必须以下面的行开始（必须放在文件的第一行）：

```sh
#!/bin/sh
```

符号`#!`用来制定执行该文件的程序。在这个例子中我们使用`/bin/sh`来执行程序。

# 注释

与`Python`的注释语法相同，以`#`开始的句子表示注释，直到这一行的结束。

# 变量

在`Shell`编程中，所有的变量都由字符串组成，并且**不需要对变量进行声明**。

要赋值给一个变量，可以这样写：

```sh
a=”hello world”
echo “A is:”

# 以下两种输出方法皆可
echo $a
echo ${a}
```
